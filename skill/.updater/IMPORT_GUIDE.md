# Trae 技能批量导入指南

## 简介
本文档详细说明如何通过压缩包批量导入技能到 Trae IDE。

## 准备工作
1. 确保你已经运行了技能更新工作流
2. 工作流会自动生成 `skills_all.zip` 文件
3. 该文件包含了所有最新的技能

## 导入步骤

### 方法1：通过设置界面导入
1. 打开 Trae IDE
2. 点击左侧边栏的设置图标（齿轮图标）
3. 在设置菜单中，选择 "规则和技能"
4. 点击 "技能模块"
5. 点击 "创建" 按钮
6. 在弹出的对话框中，选择 "上传 Skill 文件"
7. 浏览并选择生成的 `skills_all.zip` 文件
8. 点击 "确定" 按钮
9. Trae 会自动解压并加载所有技能

### 方法2：使用 .trae/skills/ 目录
1. 在项目根目录创建 `.trae/skills/` 目录
2. 将 `skills_all.zip` 文件解压到该目录
3. Trae 会自动解析并加载这些技能

## 故障排除

### 常见问题
1. **导入失败**
   - 检查压缩包是否完整
   - 确保压缩包不超过 10MB
   - 尝试重新生成压缩包

2. **技能加载不完整**
   - 检查每个技能目录是否都有 SKILL.md 文件
   - 确保 SKILL.md 文件格式正确
   - 尝试分批导入

3. **命名冲突**
   - 如果出现命名冲突，Trae 会提示你
   - 可以选择覆盖现有技能或重命名后再导入

## 自动更新机制

技能包会通过 GitHub Actions 工作流自动更新：
1. 每周日凌晨 2 点（UTC）自动触发
2. 也可以手动触发工作流
3. 更新后会自动生成新的 `skills_all.zip` 文件
4. 你可以下载最新的压缩包进行导入

## 注意事项

1. **定期更新**：建议定期导入最新的技能包，以获取最新功能
2. **备份**：在导入前，建议备份现有的技能配置
3. **版本控制**：技能包会随项目一起版本控制，确保可追溯性
4. **权限**：确保你有足够的权限在 Trae 中导入技能

## 联系支持

如果遇到任何导入问题，请：
1. 检查工作流日志获取详细信息
2. 参考 Trae 官方文档
3. 联系 Trae 支持团队
